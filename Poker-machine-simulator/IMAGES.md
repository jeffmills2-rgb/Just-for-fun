# Theme art manifest

The machine uses a single game, **Mathematics Growth Adventure**, with all art in
`assets/themes/maths/`. Drop **PNG** files in using the exact names below.
Anything missing falls back to the built-in art, so you can add pieces one at a
time.

- `background.png` — cabinet background (any size; 1600×1000 or similar looks best). Sits behind the machine.
- `<ID>.png` — a reel symbol. Recommended square (e.g. 512×512), transparent background, centred.

## Mathematics Growth Adventure  (`assets/themes/maths/`)

| File | Role |
|------|------|
| `background.png` | Cabinet background |
| `STAR.png` | Reel symbol (highest paying) |
| `DIAMOND.png` | Reel symbol |
| `CIRCLE.png` | Reel symbol |
| `TRIANGLE.png` | Reel symbol |
| `A.png` | Reel symbol |
| `K.png` | Reel symbol |
| `Q.png` | Reel symbol |
| `J.png` | Reel symbol (lowest paying) |
| `ORB.png` | Feature orb (scatter / Hold & Spin trigger) |

Swapping any image just means replacing the PNG of the same name — the theme
styling (NSW navy / red / white) and game maths are independent of the artwork.
